# Кейс: кластеризация белорусских акций

Цель кейса — показать, как аналитик может сократить избыточный набор активов перед портфельной оптимизацией. Вместо ручного выбора бумаг используется корреляционная структура месячных доходностей.

Ключевой методологический выбор: в кластеризации используются простые месячные доходности. Это сохранено для воспроизводимости результатов и согласованности с последующими модулями.

Практический результат — выбранная матрица цен по 7 активам, которая передаётся в CVaR-проект как фиксированная инвестиционная вселенная.
