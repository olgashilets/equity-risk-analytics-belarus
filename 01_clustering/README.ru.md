# 01. Кластеризация инвестиционной вселенной

[English version](README.md)

## Назначение

Кейс сокращает набор акций за счёт группировки активов со схожей динамикой месячных доходностей. Это первый этап пайплайна риск-аналитики: он формирует компактную инвестиционную вселенную для последующей портфельной оптимизации.

## Методология

- Нулевые и пропущенные цены считаются пропусками.
- Акции с числом наблюдений меньше 10 месяцев исключаются.
- Доходности считаются как **месячные простые доходности** через `pct_change()`.
- Расстояние между активами рассчитывается как `1 - |correlation|`.
- Используется иерархическая кластеризация методом Уорда.
- Из каждого кластера выбирается один представитель: акция с максимальной годовой простой доходностью внутри кластера.

## Как запустить

Из папки проекта:

```bash
python run_clustering.py
```
