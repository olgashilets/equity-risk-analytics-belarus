# Кейс: CVaR-оптимизация защитного портфеля

Этот модуль показывает работу с хвостовым риском. В отличие от исходной учебной версии, здесь больше нет повторной кластеризации: входная инвестиционная вселенная считается заранее заданной.

Профессиональная логика такая:

```text
selected asset prices → log returns → portfolio optimization → risk metrics → interpretation
```

Это делает кейс понятнее для работодателя: проект демонстрирует именно Expected Shortfall / CVaR, long-only ограничения, сравнение портфельных стратегий и интерпретацию хвостового риска.
