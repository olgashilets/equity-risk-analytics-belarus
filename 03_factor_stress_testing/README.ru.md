# 03. Макроэкономическое факторное стресс-тестирование

[English version](README.md)

## Назначение

Кейс оценивает, как три портфельные стратегии реагируют на макроэкономические шоки. Он дополняет CVaR-проект: CVaR измеряет неблагоприятный исторический хвост, а факторное стресс-тестирование измеряет чувствительность к конкретным макрофакторам.

## Входные данные

```text
data/input/portfolio_returns.csv
data/input/macro_factors.csv
data/input/macro_factors_source_levels.csv
```

`portfolio_returns.csv` берётся из CVaR-модуля. Макрофакторы сформированы на основе официальных и публичных макроэкономических рядов Беларуси и преобразованы в месячные модельные переменные.

## Макрофакторы

- `inflation_mom_pct` — месячное изменение ИПЦ, %.
- `usd_byn_depreciation_pct` — месячное ослабление BYN к USD, %.
- `refinancing_rate_change_pp` — изменение ставки рефинансирования, п.п.
- `industrial_prod_yoy_change_pp` — изменение годового темпа промышленного производства, п.п.

## Методология

Для каждого портфеля оценивается модель:

```text
R_portfolio,t (%) = α + β1·inflation + β2·FX_depreciation + β3·rate_change + β4·industrial_growth_change + ε
```

Диагностика и результаты:

- VIF для проверки мультиколлинеарности;
- факторные β;
- робастные p-value с указанием типа ковариационной оценки;
- сравнение фактической и модельной доходности;
- вклад факторов в стресс-сценарий.

Стресс-сценарий:

```text
+1 п.п. инфляции
+5% ослабления BYN к USD
+1 п.п. ставки рефинансирования
-2 п.п. промышленного роста
```

## Как запустить

Из папки проекта:

```bash
python run_factor_analysis.py
```

## Ограничения

Оценки факторной модели индикативны. Скрипт пробует HC3-ковариацию, но переключается на HC1, если HC3 становится нечисленной из-за наблюдений с высоким leverage. В выборке 24 месячных наблюдения доходностей портфелей, рынок малоликвидный, ставка меняется редко, большинство коэффициентов статистически незначимы. Это framework для sensitivity analysis и stress testing, а не прогнозная модель и не инвестиционная рекомендация.
